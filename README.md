This repository hosts my personal resume and portfolio website. It showcases projects, blog posts, skills, and contact information. The site is built with Astro and includes React components and TypeScript.

**Quick overview**

- **Tech:** Astro, TypeScript, React, Vue Js

**What's in this repo**

- `src/pages/` — site pages and routes
- `src/components/` & react components. Potentially will add VueJs and Solid components as well
- `src/content/` — Markdown/MDX content (blog posts, project pages)

Getting started

Install dependencies and run the dev server (using pnpm; npm/yarn also work):

```bash
pnpm install
pnpm dev
```

Build for production and preview locally:

```bash
pnpm build
pnpm preview
```
preview changes locally with `pnpm dev` before committing

Features:

Based on Astro Js blog template

- ✅ 100/100 Lighthouse performance
- ✅ SEO-friendly with canonical URLs and OpenGraph data
- ✅ Sitemap support
- ✅ RSS Feed support
- ✅ Markdown & MDX support
